##### **使用var声明局部变量**

```java
var 变量名称 = 值;
```

在这里var相当于是一种动态类型，编译器会根据我们赋给var的值来推断出var是什么类型的数据，有点像c++中的模板，因此，使用var的时候，必须要给var所定义的变量赋值；

同时也要注意，var不能用于声明成员变量，也不能作为方法的返回值

##### 位运算符

|      运算      |           运算符            | 运算符类型 |                          运算符法则                          |       结果精度       |
| :------------: | :-------------------------: | :--------: | :----------------------------------------------------------: | :------------------: |
|  “按位与”运算  |              &              | 双目运算符 |                   与运算：两数同为1，才为1                   | 与精度高的操作数相同 |
|  “按位或”运算  |             \|              | 双目运算符 |                   或运算：两数同为0，才为0                   | 与精度高的操作数相同 |
| “按位取反”运算 |              ~              | 单目运算符 |                   取反操作：1改为0，0改为1                   |                      |
| “按位异或”运算 |              ^              | 双目运算符 |             异或运算：两数相同时，才为0，否则为1             | 与精度高的操作数相同 |
|    移位运算    |          <<(左移)           |            | 左移：比如1100<<2，得到的结果就是1100 00，即将左边的二进制数字向左移动几位，就在右边加上几个0 |                      |
|                |          >>(右移)           |            | 右移：比如101100>>1，结果就是110110，比如011011>>1，结果就是001101；即根据位数摒弃右边几位，然后根据原先二进制数的最高位（0或1）在左边补上与最高位相同的数字 |                      |
|                | >>>(无符号            右移) |            |      与右移不同的是，无论最高位是0还是1，都在左边补上0       |                      |

移位可以实现整数除以或者乘以$2^n$的效果，例如，y<<2与y*4的效果相同，y>>1的结果与y/2的效果相同，总之，一个数向左移n位，就是将这个数乘以$2^n$；一个数向右移n位，就是将这个数除以$2^n$



##### 三元运算符

格式

```java
条件式 ? 值1:值2;
```

运算法则是：若条件式的值位true，则整个表达式取值1，否则取值2，例如：

```java
boolean b=20<45 ? true:false;

##### switch语句

```java
switch(表达式){
case 常量值1:
	语句块1
	[break;]
...
case 常量值n:
	语句块n
	[break;]
default:
	语句块n+1
	[break;]
}

//另一种写法(java14版本)
switch(表达式){
case 常量值1，常量2...->语句块;
...
case 常量值1，常量2...->语句块;
default->语句块;
}
```

该语句用于检测一个变量是否符合某个条件，如果不符合，在用另一个值来检测，以此类推

**运行法则：**

switch语句首先计算表达式的值，如果表达式的计算结果和某个case后面的常量值相同，则执行该case语句后的若干条语句，直到遇到break语句。此时，如果该case语句中没有break语句，则将继续执行后面case中的若干条语句，直到遇到break语句。若没有一个常量的值与表达式的值相同，则执行default后面的语句。default语句为可选的，如果它不存在，且switch语句中表达式的值不与任何case的常量值相同，那么switch语句不做任何处理

**注：**同一条switch语句，case的常量值必须互不相同；case语句后面的常量表达式的值可以为整数，但绝不可以是非整数的实数

byte、short、char、int、String、enum这些类型的常量允许使用

##### Arrays接口

###### 填充、替换数组元素

* ```java
    Arrays.fill(int[] a,int value);//将整个数组全部填充value
    ```

a：要进行元素填充的数组

value：要进行填充的值

* ```java
    Arrays.fill(int[] a,int fromindex,int toindex,int value);//指定范围进行数字填充
    ```

a：要进行填充的数组

fromindex：填充范围的左端点（包含）

toindex：填充范围的右端点（不包含）

value：填充的数值

若fromindex=toindex，则填充范围为空



###### 对数组进行排序

```java
Arrays.sort(object);
```

Java中的String类型数组的排序算法是根据字典编排顺序排序的，因此数字排在字母前面，大写字母排在小写字母前面



###### 数组的复制

* ```java
    Arrays.copyOf(arr,int newlength);//从头开始复制
    ```

arr：要进行复制的数组

newlength ：int型常量，指复制后的新数组的长度。如果新数组的长度大于原本的数组，则用0填充（int型的数组用0填充，char数组用null填充），如果新数组的长度小于原数组，则会从第一个数字开始截取，直至满足新数组的长度

* ```java
    Arrays.copyOfRange(arr,int fromindex,int toindex);//选定范围进行复制
    ```

选定范围进行复制，范围依旧是左闭右开



###### 数组的查询（binary方法，使用二分法进行查找，用这个方法之前<u>**一定要先进行排序**</u>）

* ```java
    //整体查找
    Arrays.sort(Object[] a);//使用这个方法之前一定要进行排序
    Arrays.binarySearch(Object[] a,Object key);//返回要查找的值（key）的下标
    ```

* ```java
    //部分查找
    Arrays.sort(Object[] a);
    Arrays.binarySearch(Object[] a,int fromindex,int toindex,int key);//指定范围进行查找，范围依旧左闭右开
    ```

第一种方法是对整个数组进行查找，第二种方法是对数组中的某一个范围进行查找

无论哪种方法，查找成功时，返回键的索引；没找到时的返回值都严格遵循 **`-(插入点) - 1`** 这个公式

**而插入点就是**：**如果你想把当前要查找的元素插入到一个已排序的数组中，并且保证插入后数组仍然有序，那么这个元素应该放的位置索引。**



##### 数组排序算法

###### 冒泡排序（相邻两个元素进行比较，较小的数在前，较大的数在后）

```java
public void Bubblesort(int[] array){
	for(int i =1;i<array.length;i++){//控制循环的总次数，一般为 数组长度-1
		for(int j=0;j<array.length-i;j++){//用来比较相邻两个元素的大小,每一趟换下来基本上就会有一个数确定下来，循环的次数就可以根据i进行减少
			if(array[j]>array[j+1]){//如果满足条件就交换位置
				int temp = array[j];
				array[j] = array[j+1];
				array[j+1] = temp;
			}
		}
	}
	showArray(array);
}
```



###### 直接选择排序

每一趟直接从待排序的数据元素中选出最小（或者最大，根据排序的升序还是降序决定）的元素，顺序的一次添加在未排序序列的后面

```java
public void Selectsort(int[] array){
	int index;
	for(int i=1;i<array.length;i++){
		index=0;//用来定义未排序序列中最大值的索引
		for(int j =1;j<=array.length-i;j++){
			if(array[j]>array[index]){
				index = j;//找出待排序序列中的最大值的索引
			}
		}
        //将未排序序列的最后一位与array[index]即最大值进行排序
		int temp = array[array.length-i];
		array[array.length-i] = array[index];
		array[index] = temp; 
	}
}
```
